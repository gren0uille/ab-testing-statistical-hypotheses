# A/B-тестирование и проверка статистических гипотез

## О проекте
Два независимых статистических исследования: A/B-тест нового интерфейса
интернет-магазина и проверка гипотезы о региональных различиях в активности
пользователей сервиса.

## Задачи
- Провести A/B-тест нового интерфейса и оценить статистическую значимость изменений
- Обеспечить корректность сплитования — выявить и исключить загрязнённые данные
- Проверить гипотезу о различиях в активности пользователей между регионами

## Ключевые результаты
- Выявил и исключил **887 пользователей**, одновременно участвовавших в нескольких
  экспериментах — это предотвратило искажение метрик при оценке результатов теста
- Провёл A/B-тест нового интерфейса на **9 963 пользователях**: обнаружен
  статистически значимый рост конверсии в покупку (p-value = 0.028), однако
  он не достиг целевого порога ТЗ в 3 п.п. — рекомендовано не внедрять фичу
- Проверил гипотезу о различиях в активности пользователей между Москвой и
  Санкт-Петербургом (**8 540 пользователей**) с помощью t-теста Уэлча:
  статистически значимой разницы не обнаружено (p-value = 0.34) —
  обоснован отказ от региональной сегментации
